Avant de commencer, lancez la commande suivante pour installer l'environnement Flask :
python3 -m venv .venv 
OU selon votre python :
python -m venv .venv

- Activez l'environnement virtuel avec la commande suivante : .venv/Scripts/activate
>>> Vous obtiendrez une ligne similaire à (avec vos noms de dossier) : (.venv) PS C:\Users\YanM1\Desktop\works\ia-imagen>

- Ensuite, exécutez la commande suivante dans le terminal à la racine :
pip install -r requirements.txt

Pour pouvoir lancer l'application, il faut exécutez la commande ci-dessus AINSI qu'activer l'environnement virtuel
- Exécutez la commande suivante : python app.py
